Operation Code

Our mission is to empower the military community to learn software development, enter the tech industry, and code the future. With that said, everyone is welcome to participate in these meetups and learn to code or provide mentorship!


Join the slack community by registering at Operation Code website.

Meetups

  • Goal: To get participants mentorship and real-world practical experience working on an open source software project within a team.
  • Operation Code: Hampton Roads meetups fall on the third Thursday of every month. See the meetup page for more information.
